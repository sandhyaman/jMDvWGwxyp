# Python学生就业管理系统

## 核心功能
- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理
- 学生教育经历
- 学生项目经历
- 学生投递记录
- 修改个人信息、密码等

## 适用场景
- 教育机构管理学生就业信息
- 企业招聘信息管理
- 学生求职信息管理

## 技术特点
- 使用Python语言和Django框架进行开发
- 前端技术：HTML、CSS、JavaScript、jQuery
- 后端技术：Python、django
- 数据库：MySQL
- 支持环境：Python3.0以上、MySQL8.0
- 开发工具：Pycharm、Navicat
